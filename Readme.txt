Book Recommendation System
Objective: 
To develop a user-friendly book recommendation system that suggests books based on user preferences and reading history, enhanced with real-time book information and reviews fetched from an external API.
Description: 
The aim is to create a simple book recommendation system that utilizes user ratings and preferences to provide personalized book suggestions. 
The system will use Python for data processing, Postgres for database management, and Faker for generating synthetic data. 
  Additionally, it will use API requests to fetch real-time book information and reviews.

User to access to access the Book Recommendation system using the link : http://127.0.0.1:5000/
